# Levva Vault

## Install dependencies

```bash
yarn
```

## Compile contracts

```bash
yarn compile
```

## Run uint tests

```bash
yarn test
```

## Run integration tests on forked mainnet

### Arbitrum

```bash
yarn test:arb
```

### Ethereum

```bash
yarn test:eth
```

## Run gas usage tests

```bash
yarn test:gas
```

## Run coverage

```bash
yarn test:coverage
```
